# Exercises

This directory is reserved for smaller practice work that is lighter than homework and shorter than an in-class activity.

Convention:

- give each exercise its own folder
- prefer stable names such as `exercise-06-8/` or `exercise-queue-basics/`
- keep the published prompt at the top of the exercise folder
- keep any starter code or support material in subfolders such as `code/` or `authoring/`
