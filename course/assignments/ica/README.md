# In-Class Activities

This directory is the source-of-truth home for in-class activities.

Keep all in-class activities together here rather than scattering them across lecture folders.

Recommended structure:

```text
course/assignments/ica/
  09/
    ica-09.md
    compose.yml
    api/
    worker/
    demo-runner/
  10/
    ica-10.md
    compose.yml
    api/
    worker/
    demo-runner/
```

Lecture notes may link to these activities, but the activity source itself should live here.
