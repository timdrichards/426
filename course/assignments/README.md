# Assignments

This directory is the source-of-truth home for assignment materials.

Recommended subdirectories:

- `homework/`: homework prompts and supporting authoring material
- `in-class-activities/`: in-class assignment authoring material
- `exercises/`: smaller practice work
- `project/`: project prompts and supporting files

The website build copies published assignment docs into `website/docs/` while keeping authored supporting material here rather than inside the website publish tree.
