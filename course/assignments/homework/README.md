# Homework

This directory is the source-of-truth home for homework prompts and supporting material.

Recommended homework structure:

- `<nn>/<nn>.md`: published homework prompt source
- `<nn>/authoring/`: instructor-facing drafts, generated handouts, rubrics, and support documents
- `<nn>/code/`: starter code or reference files when a homework needs distributed code artifacts

Example:

```text
course/assignments/homework/01/
  01.md
  authoring/
  code/
```

The website sync copies the publishable homework content into `website/docs/homework/`.
Authoring support files stay here.
