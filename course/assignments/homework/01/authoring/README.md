# Homework 01 Authoring

This directory holds authoring and generated support material for Homework 01.

Contents:

- `homework-01.md`: short assignment summary used in the authoring bundle
- `assignment-handout.md`: full student-facing handout export
- `grading-rubric.md`: grader-facing rubric
- `submission-checklist.md`: submission checklist export
- `instructor-notes.md`: instructor-only notes
- `hw01-spec.json`: assignment generation input spec
- `assignment-spec.normalized.json`: normalized generated spec
- `workload-report.md`: workload estimate report

Convention:

- keep the published homework prompt in the parent folder as `01.md`
- keep exports, generator inputs, and instructor-facing material here
