# Lectures

This directory is the source-of-truth home for lecture materials.

Recommended lecture structure:

- `slides/`: slide source and deck-local assets
- `code/`: demo/example code that may be published as downloadable archives
- `img/`: lecture-local images
- `ex/`: lecture-local exercise material when needed

The website build copies published lecture content into `website/docs/lectures/`.
In-class activities should live under `course/assignments/ica/` rather than inside lecture folders.
