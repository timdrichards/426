# Course

This directory is the source-of-truth home for authored course material.

Subdirectories:

- `lectures/`: lecture units with slides, code, and lecture-local assets
- `assignments/`: homework, in-class activities, exercises, and project material
- `readings/`: reading chapters and reading assets
- `weeks/`: week overview pages and schedule-linked summaries
- `shared/`: reusable templates and other shared assets
